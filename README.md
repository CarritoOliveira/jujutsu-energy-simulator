# Jujutsu Energy Simulator 🔥

🌐 **Acessar o Simulador:** [Jogar Agora](https://carritooliveira.github.io/jujutsu-energy-simulator/)

Um simulador interativo de batalha inspirado no universo de **Jujutsu Kaisen**. Gerencie sua Energia Amaldiçoada, execute técnicas poderosas e viva a experiência de um Feiticeiro de alto nível.

### ⚔️ Principais Funcionalidades

- **Sistema completo de Energia Amaldiçoada** com regeneração e Liberação Máxima
- **Técnicas icônicas**:
  - Ataque Básico
  - Desmantelar e Cleave
  - Corte que Corta o Mundo (com ritual de carregamento)
  - Black Flash
  - Expansão de Domínio (Santuário Malevolente)
  - RCT (Reverso da Técnica Amaldiçoada)
  - Kamino Fuga
- Sistema de **cooldowns** realistas
- **Fornalha** (acúmulo de energia para técnicas especiais)
- **HP** com dano variável (Leve, Médio, Grave e Crítico)
- **Defesa** temporária
- Interface imersiva com vídeo de fundo temático

### 🛠️ Tecnologias Utilizadas

- **HTML5**
- **CSS3** (com animações e design responsivo)
- **JavaScript Vanilla** (lógica completa do jogo)

### 🎯 Objetivo do Projeto

Desenvolvido para praticar:
- Manipulação avançada do DOM
- Gerenciamento de estados complexos
- Lógica de temporizadores (cooldowns, intervals)
- Criação de interfaces interativas e imersivas
