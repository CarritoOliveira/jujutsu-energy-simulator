# Jujutsu Energy Simulator

🌐 **Acessar o Simulador:** [Jogar Agora](https://carritooliveira.github.io/jujutsu-energy-simulator/)

Um simulador interativo de batalha inspirado no universo de **Jujutsu Kaisen**. Gerencie sua Energia Amaldiçoada, execute técnicas poderosas e lute com estilo.

### ⚔️ Funcionalidades
- Sistema completo de Energia Amaldiçoada (com regeneração)
- Diversas técnicas icônicas:
  - Ataque Básico
  - Corte que Corta o Mundo
  - Expansão de Domínio
  - RCT (Reverso da Técnica Amaldiçoada)
  - Black Flash, Cleave, Dismantle e mais
- Sistema de dano variável (Leve, Médio, Grave e Crítico)
- Cooldowns realistas para cada habilidade
- Interface temática imersiva com design dark/anime
- Log de ações em tempo real

### 🛠️ Tecnologias Utilizadas
- **HTML5**
- **Tailwind CSS**
- **JavaScript Vanilla** (sem frameworks)

### 🎯 Sobre o Projeto
Desenvolvido como projeto pessoal para praticar lógica de programação, manipulação do DOM, gerenciamento de estados complexos e criação de interfaces interativas e visualmente atraentes.
